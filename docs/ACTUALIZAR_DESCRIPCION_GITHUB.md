# Cómo Actualizar la Descripción del Repositorio en GitHub

## Opción 1: Desde la Interfaz Web de GitHub (Recomendado)

1. Ve a tu repositorio: https://github.com/leranuva/portfolio
2. Haz clic en el ícono de **⚙️ Settings** (Configuración) en la parte superior del repositorio
3. O simplemente haz clic en el botón **"Edit"** que aparece al lado del nombre del repositorio
4. En el campo **"Description"**, escribe:

```
Portfolio profesional desarrollado con Laravel 12, Tailwind CSS y Alpine.js. Sistema completo con diseño moderno, responsivo, modo oscuro, gestión de proyectos destacados, habilidades por categorías y sección de contacto integrada.
```

5. Haz clic en **"Save"** o **"Update"**

## Opción 2: Descripción Corta (140 caracteres)

Si prefieres una descripción más corta:

```
Portfolio profesional con Laravel 12, Tailwind CSS y Alpine.js. Diseño moderno, responsivo, dark mode y gestión completa de proyectos.
```

## Opción 3: Usando GitHub CLI

Si tienes GitHub CLI instalado:

```bash
gh repo edit leranuva/portfolio --description "Portfolio profesional desarrollado con Laravel 12, Tailwind CSS y Alpine.js. Sistema completo con diseño moderno, responsivo, modo oscuro, gestión de proyectos destacados, habilidades por categorías y sección de contacto integrada."
```
